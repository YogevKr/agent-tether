# Contributing

## Local setup

1. Copy `.env.example` to `.env`
2. Fill the Telegram and Codex settings
3. Run:

```bash
npm test
npm run check
```

## Change scope

- Keep changes focused
- Add or update tests when behavior changes
- Update `README.md` when setup or UX changes

## Pull requests

- Explain the user-visible change
- Include the verification commands you ran
- Call out any follow-up or known limitations
